About TripSplitter

TripSplitter is a collaborative travel expense manager designed to help groups track shared expenses and simplify settlements.


Core Features:

  • Trip Dashboard with spending summary
  • Add and track shared expenses
  • Debt simplification logic (Phase 2)
  • Spending category visualization (React integration)
  • Exportable trip report

Developed as a Mini Project

Frontend implemented using HTML, CSS, and JavaScript. Backend integration planned in next phase.
